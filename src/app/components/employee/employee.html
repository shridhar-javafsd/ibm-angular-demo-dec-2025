Data Bilding in Angular

Data to and fro compoment and template can be shared in many ways. This is called as data binding.

Types of Data Binding in Angular:

  1. Interpopation {{ '{{ expression }}' }} - Component to Template
    2. - display component data in the template.
  2. Property Binding - [property]="expression" - Component to Template -
    3. - bind a component property to an HTML element property
  3. Event Binding - Template to Component
    4. - send data or trigger logic from UI to component
  4. Two Way Bilding - [(ngModel)]="property"
    5. - Combination of property + event binding.

Angular Directives

In Angular, directives are special instructions in the DOM that tell Angular how to modify the structure, appearance, or behavior of HTML elements.

Angular has three main types of directives

  • Component Directives
  • Structural Directives
  • Attribute Directives

Component Directives

Components are actually directives with a template

Structural Directives

Change the DOM structure

*e.g. ngIf, *ngFor, *ngSwitch

Attribute Directives

Change the appearance or behavior of an element

e.g. ngClass, ngStyle, ngModel
